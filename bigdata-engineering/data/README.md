## Datasets

This folder contains **synthetic CSV datasets** generated using **Mokaroo**
for academic purposes.

### Files
- `Usuario_v1.csv`: End users receiving climate alerts
- `Pais_v1.csv`: Country reference data
- `Evento_Clima_v1.csv`: Climate-related events
- `Alerta_meteorologica_v1.csv`: Meteorological alerts linked to events and locations

All datasets are small, synthetic, and included to ensure
full reproducibility of the project.
